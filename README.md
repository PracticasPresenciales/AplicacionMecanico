# Aplicacion Taller

Descripción: Esta es una aplicación de un taller desarrollada en Java 21 utilizando JavaFX. La herramienta permite administrar el flujo de trabajo básico de un taller, incluyendo la gestión de cliente, vehículos, repuestos y usuarios de sistema.
La aplicación utiliza un sistema de persistencia basado en archivos ,txt para almacenar la información sin necesidad de una base de datos externa.

La aplicación actúa como un CRUD (Crear, Leer, Actualizar, Borrar) especializado. Al iniciar, carga los datos de los archivos. .txt. Permite al mecánico registrar entradas de vehículos, asignar repuestos con precios y fechas de mantenimiento, mantener un control de los estados de reparación.
